What is audio description?

Audio description (AD) is an additional narration track that is included in a video that explains essential visual information that the listener would not be able to understand from the original audio track alone. The main purpose of audio description is to help people who are visually impaired, as it describes things such as facial expressions, diagrams, body language, and text on the screen, but it can be utilized by anyone. For example, if a chart appears with important information but no one reads it aloud, the audio description would explain the details so listeners can understand the content. Another example might be: “She enters a room with purple walls and clutter all over the floor and walks a few steps, not noticing the backpack in front of her.” By breaking down the visual information in an accessible way, audio description ensures that everyone can take in the full meaning of the scene.

What is integrated description?

James Madison University’s digital accessibility team explains that “in pre-recorded visual and audio-video media, audio narrations must describe key visual elements during natural pauses in dialogue or sound.” Integrated description takes that idea a step further; it is a way of blending important visual details right into the main narration or dialogue so the description feels like part of the story instead of something added later.

Unlike extended audio description, where the video might pause to include extra narration, integrated description makes sure those explanations are already built into what’s being said. This creates a smoother experience for people who rely on audio or text-based information to understand what’s happening on screen. For example, if a show like Stranger Things used integrated description, a character could describe the setting or mood as part of their dialogue, helping all viewers imagine the scene without interrupting the flow.
