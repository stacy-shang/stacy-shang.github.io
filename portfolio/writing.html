Microsoft Word for Student Farmers

26-page manual on navigating Microsoft Word's interface for members of the UMass Student Farm, a year-long program year-long program in which undergraduate students work cooperatively to plan and manage over 20 acres of land on a certified organic farm. My manual targets student farmers using Microsoft Word to write the Farm Handbook, an annual publication that all student farmers must contribute to creating at the end of the season. Specifically, the manual details tasks such as merging documents, using section breaks, and creating graphs and diagrams in Microsoft Word.
